Foreword

This book is an introduction to Robust Analytics for Malaria Policy, or RAMP. It is the culmination of something I first envisioned more than 14 years ago. At some point, this book may be packaged for wider distribution. For now, the intended audience is narrow: the technical team at IHME, our RAMP-Uganda team, and partners elsewhere in the world.

I had the idea of building a modular framework for simulating mosquito-transmitted pathogens sometime in 2008. By the fall of 2019, when I sat down to write this book, I was ready to put all the pieces together, but I still felt overwhelmed by the challenge. There were some technical challenges remaining, including some that were still hidden, and they would all need to fit together seamlessly. I had tried to build modular code before, more than once, but ultimately those efforts had dead-ended. (At some point, I hope to revive MASH). As I wrote this book in 2019, in R-markdown, there was too much code in the text and it became difficult to use, so I started a parallel effort to develop a code library and manage algorithm development in the background. I called it the RAMP Model Library. With a focus on algorithm development, the book languished. Getting organized finally led to the development and release of Micro-MoB, and it’s satellite package xDE-MoB. Now, after a few key insights, I’m finally getting back to writing.

I’ve been lucky to have such an amazing team. I could name several people, but I need to single out three: Sean Wu, John Henry, and Austin Carter. Without their talents and hard work, this project would have stalled out a long time ago. The full list is found in the Acknowledgements.
